# ✈️ Flight Price Analysis (India)

## 📌 Project Overview

This project analyzes flight booking data of various airlines in India. The dataset contains detailed information about flights such as source & destination cities, departure & arrival times, duration, and ticket prices.

The main goal of this project is to extract meaningful insights and understand how different factors affect flight prices.

---

## 📊 Dataset Description

The dataset includes the following features:

* **Airline** – Name of the airline
* **Source City** – Departure city
* **Destination City** – Arrival city
* **Departure Time** – Time of departure
* **Arrival Time** – Time of arrival
* **Duration** – Total flight duration
* **Stops** – Number of stops
* **Class** – Economy / Business
* **Price** – Ticket price

---

## 🎯 Objectives

* Analyze how price varies with airlines
* Compare ticket prices across different routes
* Study the impact of stops on ticket price
* Understand how departure time affects pricing
* Identify cheapest and most expensive routes

* Business class tickets are much more expensive than economy
